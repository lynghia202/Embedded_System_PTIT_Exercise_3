# Embedded_System_PTIT_Exercise_3
## Giới thiệu
Bài tập 3 - Ngắt ngoài EXTI
## Yêu cầu:
- Cấu hình ngắt ngoài cho 1 nút nhấn. Khi nhấn nút, trạng thái của LED sẽ đảo ngược.
- Trong quá trình đó 1 LED khác vẫn nhấp nháy với chu kỳ 1Hz
## Phần cứng sử dụng
- STM32F103C8T6
- LED
- Button
